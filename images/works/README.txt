# 作品封面图片

此文件夹用于存放创作作品和约稿的封面图片。

## 文件命名规范

格式：`{作品ID}.jpg`

示例：
- `work-1.jpg` - 作品1封面
- `work-2.jpg` - 作品2封面
- `commission-1.jpg` - 约稿1封面
- `commission-2.jpg` - 约稿2封面
- ...

## 图片要求

- 格式：JPG / PNG / WebP
- 建议尺寸：宽度 600px 以上
- 建议比例：4:3（作品）、3:4（竖图约稿）
- 文件大小：建议不超过 1MB

## 使用说明

将作品封面图片放入此文件夹并按命名规范命名后，作品列表页和约稿页将自动显示对应封面图片。
