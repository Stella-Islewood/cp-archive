# 角色图集

此文件夹用于存放角色的多张展示图片。

## 文件命名规范

格式：`{角色ID}-{序号}.jpg`

示例：
- `lion-1.jpg` - Lion 第1张图
- `lion-2.jpg` - Lion 第2张图
- `lion-3.jpg` - Lion 第3张图
- `lion-4.jpg` - Lion 第4张图
- `mio-1.jpg` - Mio 第1张图
- `lucas-1.jpg` - Lucas 第1张图
- `mia-1.jpg` - Mia 第1张图
- ...

## 图片要求

- 格式：JPG / PNG / WebP
- 建议尺寸：宽度 800px 以上
- 建议比例：4:3 或 16:9
- 文件大小：建议不超过 1MB
- 每个角色建议 4 张图

## 使用说明

将图片放入此文件夹并按命名规范命名后，档案页的角色图集区域将自动显示对应图片，点击可放大查看。
